# 🚀 Projet de Fin de Semestre — CampusOps

Le code complet, l'architecture, la documentation et le fichier Docker du projet de fin de semestre se trouvent directement sur son propre dépôt GitHub officiel. 

👉 **[Cliquez ici pour accéder au Dépôt Original CampusOps](https://github.com/Hamza00-1/CampusOps-)**

### Ce que vous y trouverez :
1. **Code Source Complet** : Backend (Node.js/Express) et Frontend (React.js).
2. **Infrastructure Docker** : `Dockerfile` pour le backend/frontend, ainsi que le `docker-compose.yml` complet pour orchestrer la base de données PostgreSQL et le cache Redis.
3. **Documentation** : Le README détaillé explique le projet, son architecture, ses fonctionnalités (RBAC, envois d'emails SMTP, intégration du Bot Telegram) et la procédure de lancement exacte en une commande.

*(Note : Un rapport détaillé se trouve également dans ce dossier sous le nom `Rapport_Projet.md` pour justifier les choix architecturaux et techniques liés au module).*
